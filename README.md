# homebrew-tap

[StackFusion's](https://github.com/stackfusion) Homebrew tap.

## Install

```sh
brew tap stackfusion/tap
brew install cage
```

## Upgrade

```sh
brew upgrade cage
```

## Tools

- [cage](https://github.com/stackfusion/cage) – CLI utility for safely working with untrusted projects on macOS by isolating them in virtual machines
